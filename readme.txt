== Description ==

UI StarLinux è un tema WordPress minimale e modulare ispirato alla filosofia del software libero.
Progettato per offrire un avvio veloce e leggero, leggibile e privo di distrazioni, integra principi di
usabilità, accessibilità e coerenza visiva tipici dell’ecosistema Linux.

Il tema fornisce:
- una layout base wp essenziale e performante
- componenti modulari facilmente estendibili
- un approccio “no bloat”, senza dipendenze superflue
- un design coerente con l’estetica StarLinux
- documentazione e sviluppo aperto tramite GitHub

Repository ufficiale:  
https://github.com/starlinux-gh
